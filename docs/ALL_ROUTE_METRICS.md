## 排名规则

排名依据是四折主体隔离测试预测合并后的 **Balanced Accuracy（BA）**，不是单折最好值，也不是四折指标的简单平均。固定阈值为 0.5。

`BA = (Recall + Specificity) / 2`。它同时衡量跌倒检出率和日常活动识别率，比只看 Accuracy 更能避免类别数量或预测倾向造成的偏差。BA 完全相同的路线并列排名。

## 指标说明

- Accuracy：全部 160 个测试视频中预测正确的比例。
- Precision：预测为跌倒的样本中，真正跌倒的比例。
- Recall/Sensitivity：79 个真实跌倒中成功检出的比例；越高越少漏报。
- Specificity：81 个 ADL 中正确排除的比例；越高越少误报。
- F1：Precision 与 Recall 的调和平均。
- TP/TN/FP/FN：四折测试预测合并后的原始混淆矩阵计数。

机器可读原始值见 `results/benchmark_e300_full_summary.csv` 和 `results/benchmark_e300_full_summary.json`。
